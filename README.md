## 第一次进入开发请看这里
- 如何克隆（下载）仓库到本地
  - 在网页仓库的右上角“code”处，选择“Open with Github Desktop”。选择要存放的本地路径
  - 或者直接在本地打开Github Desktop，克隆仓库`https://github.com/Daybreak-of-Teyvat-Studio/Daybreak-of-Teyvat.git`
- 一些推荐配置
  - 打开.git/config文件（.git是隐藏文件夹，看不到的话可以百度如何查看隐藏文件夹），在末尾添加。
    ```
    [pull]
        rebase = true
    ```
